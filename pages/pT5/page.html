Relaciones de equivalencia

Sean a y b números enteros, y m mayor que cero.
Relación simétrica
a ≡ b (mod m) <=> b ≡ a (mod m)

Relación reflexiva
a ≡ a (mod m)

Relación transitiva
si a ≡ b (mod m) y b ≡ c (mod m)
Entonces
a ≡ c (mod m)


Propiedades

Si a ≡ b (mod n) y c ≡ d (mod n) entonces:

1. Para todo par de enteros r y s, ar + cs ≡ br + ds (mod n)
2. a + c ≡ b + d (mod n).
3. a - c ≡ b - d (mod n).
4. a c ≡ b d (mod n).
5. Para todo entero positivo k, a k≡bk (mod n).
6. Para todo entero r,
7. a + r ≡ c + r (mod n).
8. Para todo entero r, ar ≡ br (mod n).

Introducción a las congruencias lineales

En Álgebra se estudian detalladamente las ecuaciones polinómicas y sus soluciones. En forma análoga podemos estudiar las congruencias polinómicas. En este estudio consideramos únicamente polinomios f(x) con coeficientes enteros.

Tomado y adaptado de Jiménez et al. (2004, p. 100).

Las congruencias lineales se escriben así: ax ≡ b (mod n) y denotan un polinomio de grado uno de la forma ax + my = b.


Introducción a las congruencias lineales

Ejemplo 4

Resolver la congruencia lineal
40X ≡ 36 (mod 19)
ya que el
mcd (40,19) = 1 entonces 1|36
por lo tanto la congruencia tiene una solución.
Como el
mdc(19,4)=1 entonces
10X ≡ 9 (mod 19)
de acuerdo que
10-19 = -9 => -9X ≡ 9 (mod 19)
ya que el
mdc(19,9)=1 => -X ≡ 1 (mod 19)
se multiplica por -1 toda la congruencia
X ≡ -1 (mod 19)
de acuerdo a que
-1+19 = 18 => X ≡ 18 (mod 19)
por lo tanto la única solución incongruente a la congruencia lineal 40X ≡ 36 (mod 19) es 18.

Teorema 3

La congruencia lineal ax ≡ b (mod m) tiene solución si y solo si d | b, donde
d = (a, m).
La concruencia lineal también se puede expresar como ax + my = b.
Si la congruencia tiene solución, entonces tiene exactamente d soluciones incongruentes. Además, si x0 es una solución entonces,
x0 + k(m/d)
es solución para todo entero
k = 0,1,2,3,4,...,d-1.

Ejemplo 5

Resolver la congruencia lineal 4X ≡ 20 (mod 8)
ya que el mcd (4,8) = 4 entonces 4|20
por lo tanto la congruencia tiene cuatro soluciones.
como el mdc(4,20)=4 entonces X ≡ 5 (mod 2)
entonces X0=5
n=2 y d = (1,2) =1
entonces las soluciones estan dadas por
5 + 2k con k perteneciente a los enteros.
